Site

Os principais comandos para criar um site

Para você criar um site simples em html, você deve começar com a estrutura básica do HTML. O código inicia com !DOCTYPE html, que faz que o documento no formato HTML5. Depois, você utiliza a tag html para colocar todo o conteúdo da página. Dentro da tag html, há dois comandinhos principais: head e body. A seção head trm informações sobre o documento, como o título da página, feita pela tag title Este título se acha na aba do navegador.

A seção body é onde você coloca o conteúdo visível da página, como cabeçalhos, parágrafos, fotos, texto etc. Os textos são criados com as tags h1, h2, etc., sendo h1 o mais proeminente e h6 o menos usado. Os parágrafos são definidos pela tag p. Para listar itens, você pode usar ul para listas não ordenadas e ol para listas ordenadas, com cada item de lista sendo indicado pela tag li. Para adicionar links, você utiliza a tag a, especificando o URL de destino no atributo href. Para incluir imagens, você usa a tag img, com o atributo src indicando o caminho da imagem e alt fornecendo uma descrição alternativa.

Se você precisar criar tabelas, usa-se a tag table, com tr para definir linhas, th para cabeçalhos de colunas e td para células de dados. Finalmente, para formularios, a tag form é usada, podendo incluir campos de entrada (input) e botões. O atributo action da tag form define para onde os dados serão enviados, e o atributo method especifica o método de envio, como post ou get. Esses são os principais e fundamentais comandos para criar uma página HTML. Com esses elementos, você pode estruturar e organizar o conteúdo do seu site do seu desejo
